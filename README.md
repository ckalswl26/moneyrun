# 🏃‍♀️ MoneyRun 머니런

**머니런(MoneyRun)**은 친구, 연인, 스터디 그룹이 함께 금융 목표를 세우고 달성하는 **AI 기반 공동 금융 챌린지 앱**입니다.

기존 가계부 서비스가 개인의 소비 기록과 절약에 집중했다면, 머니런은 그룹 목표, 친구 찌르기, 미션, 배지, 랭킹, AI 코칭 등 **게이미피케이션 요소**를 활용해 사용자가 돈 관리를 포기하지 않도록 돕습니다.

---

## 💡 서비스 소개

혼자 하는 저축은 쉽게 포기할 수 있습니다.  
머니런은 “함께 달성하는 금융 목표”에 초점을 맞춰, 친구들과 목표를 공유하고 서로 응원하거나 독려하며 꾸준한 금융 습관을 만들 수 있도록 설계되었습니다.

예를 들어 여행 자금 모으기, 배달비 줄이기, 청약통장 납입 유지, 커플 결혼자금 마련 같은 목표를 그룹 단위로 설정할 수 있습니다.

AI 코치는 사용자의 소비 패턴과 목표 달성률을 분석해 실패 위험도를 알려주고, 오늘 실천할 수 있는 절약 미션을 추천합니다.

## 🚀 주요 기능

### 👥 회원가입 / 로그인
- Django Token 기반 인증
- 사용자별 프로필 관리
- 프로필 이미지 업로드 기능

### 🏦 그룹 금융 목표 관리
- 여러 개의 그룹 생성 가능
- 그룹별 목표 금액, 기간, 미션 설정
- 그룹원별 저축 현황 확인
- 초대 링크를 통한 그룹 참여

### 💬 친구 찌르기
- 입금을 잊은 친구에게 “콕 찌르기” 알림 생성
- 그룹 피드에 독려 메시지 기록
- 공동 목표 달성을 위한 소셜 책임감 강화

### 🤖 AI 코치
- 그룹 목표 달성률 분석
- 실패 위험도 계산
- 소비 패턴 기반 맞춤형 코칭 제공
- “오늘 얼마를 아끼면 목표 달성률이 올라가는지” 안내

### 🎮 챌린지 / 미니게임
- 금융 퀴즈 정답 시 1원 적립
- 미니게임 성공 시 1원 적립
- 게임처럼 재미있는 금융 습관 형성

### 🏅 게이미피케이션
- 목표 달성률
- 배지
- 런 포인트
- 그룹 피드
- 랭킹
- 연속 성공 기록 확장 가능

### 🟡 카카오톡 초대
- 그룹 초대 링크 생성
- 카카오톡 공유 버튼 제공
- 카카오 JavaScript 키가 없을 경우 링크 복사 방식으로 대체

### 🏦 카카오뱅크 연동 데모
- 카카오뱅크 계좌 연결 UI 제공
- 실제 API 연동 전 시연용 기능 구현
- 향후 오픈뱅킹 또는 금융 API 연동 확장 가능

## 🎯 차별점

머니런은 단순한 가계부나 모임통장이 아닙니다.

기존 금융 앱이 개인의 지출 관리에 집중한다면, 머니런은 **관계 기반 금융 목표 달성**에 집중합니다. 사용자는 친구들과 함께 목표를 만들고, 서로의 진행 상황을 확인하며, 게임처럼 금융 습관을 지속할 수 있습니다.

특히 친구 찌르기, 그룹 피드, 1원 적립 게임, AI 코치 기능을 통해 “혼자 관리하는 돈”이 아니라 **함께 완주하는 금융 챌린지**를 제공합니다.

## 🧑‍💻 실행 방법

### Backend

```bash
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python3 manage.py migrate
python3 manage.py seed_data
python3 manage.py runserver
```

### Frontend

```bash
cd frontend
cp .env.example .env
npm install
npm run dev -- --host 0.0.0.0 --port 5178
```

### 접속 주소

```text
Frontend: http://localhost:5178/
Backend API: http://127.0.0.1:8000/api/dashboard/
```
